# Study Packs

Study packs contain text-specific policy and learner state. The generic math agent skills read `studies/active.md`, then use the active pack for source priority, progress, workflow, and exercise selection.

Current packs:

- `judson-aata/`: Thomas W. Judson, *Abstract Algebra: Theory and Applications*.

To add another mathematics text, create a new directory with at least:

- `profile.md`
- `source-priority.md`
- `workflow.md`
- `state/progress.md`

Then update `studies/active.md` when that pack should be the default.
